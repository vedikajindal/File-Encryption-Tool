# Java-Based Secure File Encryption Tool

This project is a **Java-based Secure File Encryption Tool** developed to provide modern and highly secure encryption method which is used worldwide.  
The system uses **Object-Oriented Programming (OOP)** principles to ensure modularity, scalability, and clean design.

---

## Objective

The objective of this project is to:

- User login system with admin and normal users 
- Generate encryption keys 
- Encrypt file 
- Decrypt files 
- Create new text files 
- View file details 
- User management (Admin only) 
- Security log system
---

## OOP Concepts Used

- Classes and Objects  
- Encapsulation 
- Abstraction 
- Polymorphism & Inheritance 
- Composition (HAS-A Relationship) 
- File Handling 
- Exception Handling 
- Collections 
- Java Security API 
- Streams and Buffers 
- Command-Line Interface (CLI)
